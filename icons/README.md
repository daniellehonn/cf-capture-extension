# Icons

This directory should contain the following icon files:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using any image editor. The icon should represent a video capture/camera theme.

For a quick start, you can:
1. Use an online icon generator like https://www.favicon-generator.org/ or https://www.designbold.com/youtube-icon/
2. Create a simple SVG icon and convert it to PNG
3. Use any red YouTube-inspired design with a camera/capture symbol

Example: A red camera icon or a play button with a plus sign would work well.

**Note:** The extension will work without icons, but Chrome will show a default puzzle piece icon instead of a custom one.
